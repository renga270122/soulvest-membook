AI-powered digital memory book for couples

## Features
- Create beautiful relationship timeline
- AI weaves memories into narrative story (now using Google Gemini)
- Multiple chapters: First Date, Milestones, Challenges, Dreams
- Download as PDF keepsake
- Add photos (optional)
- Can update and add new memories anytime
- Custom branding and romantic UI
- Heart beat animation
- Rotating romantic quotes
- Social sharing and gift links

## Tech Stack
- Python
- Streamlit
- Google Gemini API (via google-generativeai)
- FPDF for PDF generation
- Pillow for image handling

## Run Locally
```
pip install -r requirements.txt
streamlit run app.py
```

## Google Gemini API Setup
- Sign up for Google Gemini API and get your API key.
- Add your Gemini API key to Streamlit secrets or environment variable as needed.
